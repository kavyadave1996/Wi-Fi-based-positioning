# Wi-Fi-based-positioning
Developing a 3D indoor localization framework for mobile robots by coupling BIM models with IEEE 802.11az Wi-Fi standards. Features include genetic algorithm (MOGA) optimization for AP placement, ray-tracing simulations for signal multipath modeling, and ResNet-18 deep learning for centimeter-level positioning accuracy.
